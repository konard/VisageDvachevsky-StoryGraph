## How to Add Your Own Assets

### Replacing Background Music

1. Download royalty-free music (OGG format preferred)
2. Place files in `assets/music/`
3. Update file references in `main.nms` if filenames differ

### Replacing Sound Effects

1. Download CC0 sound effects (WAV format)
2. Place files in `assets/sfx/`
3. Ensure filenames match script references:
   - `door_open.wav`
   - `footsteps.wav`
   - `ui_click.wav`

### Adding Voice Acting

1. Record or commission voice lines
2. Place files in `assets/voice/`
3. Enable voice in `project.json`:
   ```json
   "voiceActingEnabled": true
   ```
4. Add voice mapping for characters

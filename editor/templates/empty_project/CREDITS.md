# Asset Credits

This template contains no bundled assets. When you add your own assets, please document their sources and licenses here.

## Template Structure

- **Template Design**: NovelMind Team (CC0 - Public Domain)
- **Script Examples**: NovelMind Team (CC0 - Public Domain)

## Adding Your Own Credits

When you add assets to your project, update this file with:

```markdown
## Category

### Asset Name
- **Source**: [URL or description]
- **Author**: [Author name]
- **License**: [License type]
```

## Recommended License Types

- **CC0 (Public Domain)**: Free for any use, no attribution required
- **CC BY**: Free for any use, attribution required
- **CC BY-SA**: Free for any use, attribution required, share alike
- **CC BY-NC**: Free for non-commercial use, attribution required
- **Custom/Royalty-Free**: Check specific terms

## License Compatibility

When using multiple assets, ensure licenses are compatible:
- CC0 works with everything
- CC BY works with most licenses
- Check NC (non-commercial) restrictions carefully

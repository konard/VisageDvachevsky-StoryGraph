# Tutorial 3: Variables

Узнайте, как использовать переменные для отслеживания состояния игры и создания динамических историй.

## Что вы узнаете

- Как объявлять переменные
- Как изменять значения переменных
- Как использовать переменные в тексте
- Как создавать условия на основе переменных

## Время выполнения

⏱️ **15 минут**

## Инструкции

### Шаг 1: Изучите код

Откройте файл `variables.nms`:

```nms
// Объявление переменных
var friendship = 0
var gold = 100

scene intro {
    say Hero "У меня есть {gold} золотых монет."

    choice {
        "Подарить другу монеты" -> give_gold
        "Оставить себе" -> keep_gold
    }
}
```

### Шаг 2: Понимание переменных

**Объявление**:
```nms
var variable_name = initial_value
```

**Изменение**:
```nms
set variable_name = new_value
set score = score + 10    // Увеличение
set health = health - 5    // Уменьшение
```

**Использование в тексте**:
```nms
say Hero "Счёт: {score}"
say Hero "У меня {gold} монет и {friendship} друзей."
```

**Условия**:
```nms
if variable >= 10 {
    // Код выполнится, если условие истинно
}
```

### Шаг 4: Экспериментируйте

**Задание 1**: Добавьте новую переменную "здоровье"
```nms
var health = 100

scene check_health {
    say Hero "Моё здоровье: {health}"

    if health > 50 {
        say Hero "Я чувствую себя отлично!"
    } else {
        say Hero "Мне нужно отдохнуть..."
    }
}
```

**Задание 2**: Создайте счётчик выборов
```nms
var choices_made = 0

scene decision {
    choice {
        "Вариант 1" -> {
            set choices_made = choices_made + 1
            goto next
        }
        "Вариант 2" -> {
            set choices_made = choices_made + 1
            goto next
        }
    }
}

scene next {
    say "Вы сделали {choices_made} выборов."
}
```

**Задание 3**: Измените пороговые значения
```nms
if friendship >= 20 {  // Было >= 10
    say Friend "Ты мой лучший друг!"
}
```

## Операторы сравнения

- `==` - равно
- `!=` - не равно
- `>` - больше
- `<` - меньше
- `>=` - больше или равно
- `<=` - меньше или равно

## Арифметические операторы

- `+` - сложение
- `-` - вычитание
- `*` - умножение
- `/` - деление

## Типы данных

- **Число**: `var score = 100`
- **Строка**: `var name = "Алекс"`
- **Булево**: `var is_alive = true`

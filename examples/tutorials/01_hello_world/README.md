# Tutorial 1: Hello World

Простейший пример NM Script - выводим текст на экран.

## Что вы узнаете

- Как объявить персонажа
- Как создать сцену
- Как показать фон
- Как сделать персонажа говорящим

## Время выполнения

⏱️ **5 минут**

## Инструкции

### Шаг 1: Изучите код

Откройте файл `hello.nms` и изучите его структуру:

```nms
// Объявление персонажа
character Hero(name="Алекс")

// Первая сцена
scene intro {
    show background "bg_room"
    show Hero at center

    say Hero "Привет, мир!"
    say Hero "Добро пожаловать в мою первую визуальную новеллу!"
}
```

### Шаг 2: Что происходит?

1. **`character Hero(name="Алекс")`** - объявляем персонажа с именем "Алекс"
2. **`scene intro { ... }`** - создаём сцену с именем "intro"
3. **`show background "bg_room"`** - показываем фон
4. **`show Hero at center`** - показываем персонажа в центре экрана
5. **`say Hero "..."`** - персонаж произносит текст

### Шаг 3: Запустите

1. Откройте NovelMind Editor
2. File → Open Project
3. Выберите папку `01_hello_world`
4. Нажмите ▶ Play

### Шаг 4: Экспериментируйте

Попробуйте изменить:

- Имя персонажа: `character Hero(name="Ваше имя")`
- Текст диалога: `say Hero "Ваш текст"`
- Добавьте больше строк диалога

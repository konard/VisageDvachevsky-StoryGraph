# Tutorial 2: Choices

Узнайте, как добавить выборы игрока и создать ветвящуюся историю.

## Что вы узнаете

- Как создать меню выборов
- Как связать выборы со сценами
- Как добавить несколько персонажей
- Как использовать разные позиции персонажей

## Время выполнения

⏱️ **10 минут**

## Инструкции

### Шаг 1: Изучите код

Откройте файл `choices.nms`:

```nms
character Hero(name="Алекс")
character Friend(name="Саша", color="#D94A4A")

scene intro {
    show background "bg_room"
    show Hero at center

    say Hero "Привет! Что мне сегодня делать?"

    // Меню выбора
    choice {
        "Поговорить с другом" -> talk_scene
        "Пойти на прогулку" -> walk_scene
    }
}
```

### Шаг 2: Понимание выборов

**Синтаксис выбора**:
```nms
choice {
    "Текст варианта 1" -> scene_name_1
    "Текст варианта 2" -> scene_name_2
}
```

- `"Текст варианта"` - то, что видит игрок
- `-> scene_name` - куда ведёт этот выбор

### Шаг 3: Запустите

1. Откройте NovelMind Editor
2. File → Open Project
3. Выберите папку `02_choices`
4. Нажмите ▶ Play
5. Сделайте выбор!

### Шаг 4: Экспериментируйте

**Задание 1**: Добавьте третий вариант выбора
```nms
choice {
    "Поговорить с другом" -> talk_scene
    "Пойти на прогулку" -> walk_scene
    "Почитать книгу" -> read_scene    // Новый вариант
}

scene read_scene {
    show background "bg_library"
    show Hero at center
    say Hero "Отличная книга!"
}
```

**Задание 2**: Измените цвета персонажей
```nms
character Friend(name="Саша", color="#4AD94A")  // Зелёный
```

**Задание 3**: Добавьте больше диалога перед выбором

## Полезные цвета

- Красный: `#D94A4A`
- Синий: `#4A90D9`
- Зелёный: `#4AD94A`
- Жёлтый: `#D9D94A`
- Фиолетовый: `#D94AD9`
